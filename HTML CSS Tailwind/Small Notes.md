### HTML Semantics
Do not overuse `<div>`. Other elements include:
- section - group elements that go together visually through the page
- header
- footer
- nav
- ul
- li
- article - widget, can be stand-alone
### CSS Reset
```css
* {
	margin: 0;
	padding: 0;
	box-sizing: border-box;
}
```
