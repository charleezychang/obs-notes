### HTML Semantics
Do not overuse `<div>`. Other elements include:
- section - group elements that go together
- header
- footer
- nav
- ul
- li
- article - widget, can be stand-alone

```css
* {
	margin: 0;
	padding: 0;
	box-sizing: border-box;
}
```
